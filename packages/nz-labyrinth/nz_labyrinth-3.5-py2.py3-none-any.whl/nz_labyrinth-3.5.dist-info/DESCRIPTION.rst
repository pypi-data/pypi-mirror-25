# p3-labyrinthe
### Goal of this mini game

1. Collect objects
2. Escape from the labyrinth
3. Watch out for the guard!

Keyboard controls :

* Arrow keys to move
* Esc to quit game


### Easy installation 

```bash
pip install nz-labyrinth
```
