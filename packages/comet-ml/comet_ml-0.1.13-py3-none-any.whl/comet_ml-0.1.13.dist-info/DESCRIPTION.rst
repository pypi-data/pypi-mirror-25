Usage
-----

Installation as a pip library
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
Python2:

pip install comet-ml

Python3:

pip3 install comet-ml

force refresh:
pip install --no-cache-dir --upgrade comet-ml
pip3 install --no-cache-dir --upgrade comet-ml

Unit Tests
^^^^^^^^^^
Install pytest:
    pip install pytest

from trackml directory run:
    $ pytest
